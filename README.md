# Bank_Loan_Analysis

# 📊 Bank Loan Analysis | SQL + Power BI Project

## 🔥 Project Overview

This project focuses on analyzing bank loan data to understand lending performance, risk, and customer behavior.

The goal is to build a complete data analysis pipeline using:

* SQL for data extraction & analysis
* Power BI for visualization
* Domain knowledge for business insights

---

## 🎯 Problem Statement

Banks need to monitor loan performance using key metrics like:

* Total Loan Applications
* Funded Amount
* Amount Received
* Interest Rate
* Debt-to-Income Ratio (DTI)

This project helps in identifying:

* Good vs Bad loans
* Regional trends
* Customer behavior patterns

---

## 🛠️ Tech Stack

* SQL Server (Data Analysis)
* Power BI (Dashboard)
* Excel (Data Source)

---

## 📌 Key Features

### 1. KPI Dashboard

* Total Loan Applications

### 2. Loan Quality Analysis

## 📊 SQL Analysis

Performed complex SQL queries including:

* Aggregations (SUM, AVG, COUNT)
* Group By Analysis
* MTD & PMTD Calculations
* Loan Status Classification

Example:

```sql
SELECT COUNT(id) AS Total_Applications 
FROM bank_loan_data;
```
